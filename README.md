# LINE業務アシスタントBot

運転中などに流れてしまう夫婦間のLINE業務依頼を、自動でタスク化するBotです。

業務連絡用グループではBotが依頼を拾うだけにし、通知・完了操作・リマインドは別の「やることリスト」グループに集約できます。

## 現在できること

- 普段どおりの文章から依頼を自動判定
- 固定ルールで拾えない言い回しはCloudflare Workers AIが補完
- 業務連絡用グループではBotが返信せず、チャットを増やさない
- やることリストグループに「誤登録」「完了」「報告完了」ボタンを表示
- `一覧` で未完了タスクを表示
- 未完了がある時だけ、やることリストグループへ毎日5:00・11:00・16:00に通知
- 「報告完了」を押した時だけ、業務連絡用グループへ完了報告を送信
- 通常の依頼登録は一行表示＋クイックボタンにして、トークを埋めない
- 管理番号は内部だけで保持し、通常画面には表示しない
- `AI確認` でWorkers AIの接続状態を確認
- LINE Webhookの署名検証

## 普段の使い方

特別な書式や期限入力は不要です。

```text
揚げ餅2箱発注しておいて。
```

```text
さつまいも頼んどいて
```

```text
そろそろ芋なくなるよ
```

最後の例は固定ルールには該当せず、AIが「補充が必要な依頼」と判断します。

通知先のやることリストグループでは、必要に応じて次を操作します。

- `完了`: タスクだけ完了にする。業務連絡用グループには送らない
- `報告完了`: タスクを完了にし、業務連絡用グループへ「完了しました」を送る
- `誤登録`: タスクではなかったものを削除する

## 構成

- LINE公式アカウント / Messaging API
- Cloudflare Workers
- Cloudflare D1
- Cloudflare Workers AI
- Cloudflare Cron Triggers

## 安全設計

- 認証情報はCloudflareの暗号化Secretへ保存
- Webhook署名を検証
- Botは依頼を記録するだけで、発注・支払い・外部送信は実行しない
- 完了報告は「報告完了」を押した時だけ送信する
- AIが停止しても固定ルールによる受付は継続
- 専用Botを関係のないLINEグループへ追加しない

詳しい導入方法は `SETUP_GUIDE.md` を参照してください。
